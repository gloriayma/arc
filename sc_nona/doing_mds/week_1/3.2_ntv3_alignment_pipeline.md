# NTv3 alignment pipeline for functional / single-cell genomics data

Source: Boshar et al., "A foundational model for joint sequence-function multi-species modeling at scale for long-range genomic prediction" (InstaDeep, bioRxiv 2025-12-22). All page references below refer to the PDF in `/Users/gloria/dev/research/arc/sc_nona/papers/ntv3.pdf`.

NTv3 is a 1 Mb-context U-Net DNA language model. Pre-training is pure-DNA MLM on OpenGenome2; *post-training* adds supervised heads for ~15,889 bulk functional tracks + genome annotations across 24 animal/plant species (Sec. 2.2, p. 6; Sec. 4.6, p. 37). The paragraphs below answer the five questions and flag what is missing.

## 1. Which functional / single-cell modalities are incorporated?

**Modalities (Sec. 2.2, Fig. 3B, p. 6–7; Sec. 4.6.1, p. 37):**
- RNA-seq, CAGE (FANTOM5), ATAC-seq, DNase-seq, ChIP-seq (TFs + histone marks), GRO-seq.
- Genome-annotation labels (21 classes) — protein-coding genes, lncRNA, exon, intron, 5′/3′ UTR (split by strand), splice donor/acceptor, polyA, tissue-invariant / tissue-specific promoters and enhancers, CTCF-bound, ORFs, start/stop codons, skipped vs always-on exons (Sec. 4.6.2, p. 40).
- A **single single-cell dataset** is included: the *Drosophila melanogaster* embryogenesis scRNA-seq + scATAC-seq atlas of Calderon et al. (Sec. 4.6.1, p. 38). This is the *only* single-cell modality. There is no human/mouse scRNA, no scATAC, no Tabula Sapiens, no HCA, no Perturb-seq.
- Hi-C / 3C / micro-C: **not used**. The Discussion explicitly flags this gap: "only indirect access to 3D genome organization" (Sec. 3.1, p. 20).

Track counts by species (Sec. 4.6, p. 37): human 7,362; mouse 2,450; fly 180; Arabidopsis 1,899; rice 1,392; maize 921; wheat 776; soybean 590; cotton 319. 15 additional species contribute only annotations.

## 2. Exact preprocessing / alignment pipeline

### Reference genomes (Sec. 4.6.3, p. 41)
hg38 (human), mm10 (mouse), dm6 (fly), TAIR10, Zm-B73-REFERENCE-NAM-5.0 (maize), IRGSP-1.0 (rice), *Glycine-max-v2.1*, IWGSC (wheat), *Gossypium-hirsutum-v2.1* (cotton), plus 15 annotation-only assemblies (galGal6, danRer11, ce11, ROS-Cfam-1, ChiLan1, mRatBN7, gorGor4, etc.). No T2T.

### Human + mouse ENCODE (Sec. 4.6.1, p. 38)
- Used the **ENCODE v3 BigWig signal tracks corresponding to the exact sets included in Borzoi** [ref 7]. So the human/mouse pipeline is essentially inherited from Borzoi.
- Consortium-merged-replicate BigWigs were preferred; otherwise replicates were merged by the authors.
- Human files accessed 2025-02-24, mouse 2025-07-20.

### Human + mouse CAGE (FANTOM5) (p. 38)
- hg38 / mm10 reprocessed FANTOM5 `ctss.bed` files from `fantom.gsc.riken.jp`. Replicates identified by `experiment_name` + `donor_id`; each strand processed independently.
- **Normalization: CPM per strand per replicate**, then averaged across replicates, then converted to BigWig.

### Human + mouse GEO data (RNA-seq, ATAC-seq, DNase-seq, ChIP-seq) (p. 38)
- Raw reads aligned with **bowtie2 (default params)** to GRCh38 / GRCm38(mm10).
- BAMs coordinate-sorted with samtools.
- **CPM-normalized genome-wide coverage**; no further filtering, no duplicate removal.

### Drosophila scRNA / scATAC (Calderon et al.) (Sec. 4.6.1, p. 38)
- ~1.5 M nuclei across 0–20 h embryogenesis, neural-network age-inferred trajectory.
- Authors used the *published cell-type and developmental-age annotations* and produced **genome-wide RNA-seq and ATAC-seq coverage tracks by aggregating nuclei belonging to each annotated population** (i.e., pseudobulk per cell-type × age bin).
- Quality filter: only cell types whose NNLS RNA↔ATAC mixture coefficient > 0.1 (Calderon's score), restricted to **the 90 cell types with strongest RNA–ATAC agreement**. This yields the 180 fly tracks.
- *Missing*: exact aligner / cellranger version / cell-count per pseudobulk / UMI-handling are not specified; the paper just says "aggregating nuclei."

### Plant RNA-seq (Sec. 4.6.1, p. 38–39)
1. NCBI SRA queried per species; filtered to PAIRED, ≥2.5×10⁹ bases, ≥200 bp avg read length; only WT (mutants excluded via GPT-o3 metadata curation).
2. Genome index built with **STAR** (`--sjdbOverhang 299`, species-specific `--genomeSAindexNbases`).
3. **Trim Galore** `-quality 20 -length 20`.
4. **STAR** alignment, BAM SortedByCoordinate.
5. **`bamCoverage --normalizeUsing CPM --binSize 1`** → per-bp BigWig.

### Plant ATAC-seq + ChIP-seq (Sec. 4.6.1, p. 39–40)
1. FASTQC → Trim Galore (`-q 20 -phred 30 -stringency 3 -length 20 -e 0.1`).
2. **Bowtie2 `-X2000`** (paired-end, max fragment 2000 bp).
3. SamTools filter (`-f 2 -F 1804 -q 30`) + `fixmate -r`.
4. Picard MarkDuplicates.
5. ATAC-seq: `alignmentSieve --ATACshift` to shift to Tn5 cut site → 1-bp insertion track via `bamCoverage --normalizeUsing CPM` at **bin size 1**.
6. ChIP-seq narrow marks (TFs): reads centered (`bamCoverage --centerReads`), CPM-normalized at 1-bp.
7. ChIP-seq histone marks: full-fragment coverage, CPM at 1-bp.
8. ENCODE-style QC: bowtie2 alignment rate ≥ 0.8, organelle fraction < 0.5, PBC1 ≥ 0.5, PBC2 ≥ 1, NRF ≥ 0.5, duplication < 0.4, NFR + mono-nucleosome peaks present, read-depth ≥ species-specific threshold (scaled from ENCODE's 25 M).

### Chicken ATAC-seq + RNA-seq (Sec. 4.8.1, p. 43)
Pan et al. dataset (BioProject PRJEB55656), seven tissues. Same pipelines as plants (Bowtie2 + Tn5 shift + `bamCoverage` CPM, 1 bp). Used only in NTv3 Benchmark, not pre/post-training.

### Tomato ATAC-seq + RNA-seq (Sec. 4.8.1, p. 44)
BioProject PRJNA1228166, fruit-development tissues, same Bowtie2/`bamCoverage` pipeline.

### Ribo-seq (Sec. 4.8.1, p. 44)
Plants only (maize/rice/Arabidopsis, BioProject PRJNA637713). Trim Galore → **Bowtie2 to deplete rRNA/tRNA** → **HISAT2** to align to genome → samtools filter → `bamCoverage` CPM at 1 bp.

### Resolution / bin size
**All tracks are produced at 1 bp resolution** (`bamCoverage --binSize 1`), stored as BigWig, and consumed natively at 1 bp by NTv3 (Sec. 4.6.1 p. 39; Sec. 2.2 p. 6 — "base-resolution functional tracks").

### Track-level normalization fed to loss (Sec. 4.3.4, p. 30; Sec. 4.6.1 p. 40)
- Targets are divided by a static **per-track mean μ_t** computed over all assembled chromosomes (AlphaGenome scaling, ref [8]).
- **RNA-seq only:** sublinear power transform `x^0.75` + soft sqrt-clipping for values > 10.
- Loss is computed on the **center 37.5%** of the input window (Sec. 4.3.5, p. 31).

### How tracks are aligned to the model input
Tracks are coordinate-aligned BigWigs over the *same genome window* the model sees as DNA tokens. The functional-track head outputs a tensor of shape `(seq_len, num_tracks)` at 1 bp, so the alignment is *parallel-track / output-head* style (like Borzoi / AlphaGenome / Enformer) — **not** token-level conditioning and **not** an MSA-style stack of tracks fed as input.

## 3. How the aligned signal is used in the architecture

**Output-head style supervision** (Sec. 4.1.1 Output Heads, p. 24; Sec. 4.3, p. 29):
- One `functional_tracks_head` *per species* (linear → softplus, layer-norm beforehand). Outputs zero-padded to the max track count across species for batched compute. Human head emits 7,362 tracks at every base.
- One shared `genome_annotation_head` (21 classes × binary, focal loss γ=2.0).
- Language-model head (MLM) kept active during post-training as a regularizer.
- **Species conditioning** via Adaptive LayerNorm + zero-initialized gated residuals on every conv/transformer/deconv block (Sec. 4.1.1 Species Conditioning, p. 25). A learned species embedding modulates the entire stack. This is the *only* form of conditioning on metadata — there is no cell-type token, no assay token, no tissue token. Cell type and assay are encoded only by *which output track index* the supervision targets.
- Loss: Poisson + multinomial on tracks (down-weighted 1/5 on Poisson term), plus focal-loss on annotations, plus MLM (Eq. 5, Sec. 4.3.5, p. 31). This is the AlphaGenome / Borzoi-family multitask recipe.

**Not used:** MSA-style vertical attention, track stacking as input, auxiliary contrastive losses across tracks, Hi-C-derived 2-D targets, single-cell expression vectors as input conditioning.

## 4. Datasets, cell types, sources

- **Human/mouse:** ENCODE v3 (BigWigs matching Borzoi's set), FANTOM5 CAGE (hg38/mm10 reprocessed), GEO RNA/ATAC/DNase/ChIP-seq accessions. Cellular contexts named in human fine-tuning (Sec. 4.8.1, p. 43) include K562, HepG2, GM18861, BLaER1, MCF10A, motor neuron, ureter, Caco-2, Calu3, and selected GTEx tissues.
- **Fly:** Calderon et al. embryogenesis scATLAS, ~1.5 M nuclei → 90 cell-type pseudobulks → 180 tracks.
- **Plants** (Arabidopsis, maize, rice, soybean, cotton, wheat): SRA-mined RNA-seq + curated ATAC/ChIP. ATAC/ChIP track counts after QC are small (e.g., maize 100, *G. max* 14).
- **Chicken (NTv3 Benchmark only):** Pan et al. PRJEB55656, 7 tissues.
- **Tomato (NTv3 Benchmark, unseen species):** PRJNA1228166, fruit-development tissues.
- **Annotations:** GENCODE V44 for human; Ensembl GFF3/GTF for 15 other animal species (Sec. 4.6.2, p. 40).

Splits (Sec. 4.6.3, p. 41): human/mouse use Borzoi folds (fold3 test, fold4 val); other species use homology-based MCL clustering on 1 Mb gene-aware windows to prevent leakage.

## 5. Single-cell handling specifically

NTv3 is overwhelmingly a **bulk functional-genomics** model. The single exception is fly Calderon scRNA + scATAC, where the authors:
- Aggregate single nuclei into pseudobulks by *published cell-type × developmental age* annotation.
- Filter to 90 cell types with NNLS RNA↔ATAC mixture coefficient > 0.1 (modality-matched populations).
- Treat each resulting pseudobulk as a normal CPM BigWig track, fed identically to the bulk tracks.

There is **no sparsity-aware modeling, no per-cell loss, no scRNA latent input, no cell-type embedding**. The Discussion (Sec. 3.1, p. 20) flags this as an explicit limitation/future direction: "Incorporating additional species and richer modalities — particularly single-cell assays, perturbation datasets, and multiome measurements — would provide stronger constraints on context-specific regulatory mechanisms."

## Gaps / things the paper does not specify

- For ENCODE BigWigs inherited from Borzoi: the paper does not restate the per-assay scaling factors / soft-clip parameters; they are inherited via Borzoi's repo (Sec. 4.7.1, p. 41 mentions Borzoi-style unscaling).
- Drosophila Calderon pipeline: the paper does not state whether cellranger / STARsolo / `Cell Ranger ATAC` was used to produce per-nucleus signal before aggregation, nor how the per-bp coverage is computed from droplet data — only "aggregating nuclei."
- No per-cell UMI-deduplication strategy is documented.
- Read-length / fragment-size handling for ATAC pseudobulks in fly is not described separately from the plant ATAC pipeline.
- No description of how the model handles strand for stranded RNA-seq (CAGE is explicitly per-strand; bulk RNA-seq via `bamCoverage` default is unstranded — not stated).
- No mention of GTEx single-tissue handling beyond "selected GTEx tissues" in human fine-tuning (Sec. 4.8.1, p. 43).
- Inflation / saturation handling for highly-expressed loci beyond the `x^0.75` + soft sqrt-clip on RNA-seq is not detailed.

## One-line summary for the sc_nona project

NTv3's "single-cell" story is a **single fly atlas, pseudobulked to 90 cell types and treated as ordinary CPM BigWig tracks at 1 bp** — alignment is bowtie2 / STAR / HISAT2 to a per-species reference, output is a parallel per-species multi-track Poisson-multinomial head with species (not cell-type) conditioning via AdaLN. There is no MSA-style vertical attention over tracks and no genotype-axis variation in the supervision. Everything else is bulk ENCODE/FANTOM/GEO inherited from Borzoi's recipe.
